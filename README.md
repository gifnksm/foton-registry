# foton-registry

Official package registry for [foton](https://github.com/gifnksm/foton).

This repository contains versioned package manifests used by `foton` to resolve and install font packages.

It does not contain the font files themselves.
Each manifest describes where a package is fetched from and how it is installed.

## What this repository is for

This repository serves as the default registry for `foton`.

If you are looking for the package manager itself, see:

- [gifnksm/foton](https://github.com/gifnksm/foton)
